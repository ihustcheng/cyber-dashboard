# 🦞 赛博任务看板

> AI Agent Task Dashboard · Powered by [OpenClaw](https://openclaw.ai)

赛博（Cyber Lobster）的任务追踪可视化看板。深色赛博朋克风格，支持任务状态、优先级、阶段、执行日志的实时展示。

## 🔗 在线访问

👉 **https://ihustcheng.github.io/cyber-dashboard**

## 架构

```
MEMORY.md（真相源）→ 飞书 Bitable（数据层）→ GitHub Pages（展示层）
```

## 功能

- 📊 统计卡片（总任务/执行中/待处理/已完成/P1重要）
- 📈 总体进度条
- 🎴 三栏看板（进行中 / 待处理 / 已完成）
- 🏷️ 彩色标签（优先级 / 分类 / 阶段）
- 📝 执行日志（关键动作追踪）
- ⚠️ 阻塞原因高亮
- 🌙 深色赛博朋克主题 + 动效

## 技术栈

- 纯 HTML/CSS，零依赖，单文件
- GitHub Pages 静态托管
- 数据由赛博🦞自动更新
